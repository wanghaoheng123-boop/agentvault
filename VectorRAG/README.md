# Vector RAG References

Store massive, unstructured reference materials here.

**Allowed Formats:**
- `.pdf` (Research papers, extensive documentation)
- `.md` (Massive README files from GitHub repos)
- `.txt` (Raw data dumps, external API payloads)

**Rule:** Do NOT ask the LLM to read the entire contents of files in this directory simultaneously. Use embedding-based similarity search (or agentic RAG tools) to query these documents as needed.
